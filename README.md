# 🌍 Global Terrorism Analysis (1970–2017)
### Exploratory Data Analysis using Python | UNGTA Dataset

---

## 📌 Overview
This project performs a comprehensive **Exploratory Data Analysis (EDA)**
on the United Nations Global Terrorism Analysis (UNGTA) dataset containing
**180,000+ terrorist incidents** across **205 countries** from **1970 to 2017**.
The goal is to uncover global terrorism patterns, trends, and risk factors
through data cleaning, feature engineering, and rich visualizations.

---

## 🛠️ Tech Stack
- **Python** · Pandas · NumPy · Matplotlib · Seaborn · Plotly
- **Platform:** Google Colab 

---

## 📊 What's Inside
- ✅ Missing value treatment strategy
- ✅ Outlier detection using 3×IQR capping
- ✅ Feature engineering (casualties, decade, mass-attack flag)
- ✅ 12 visualizations — trends, heatmaps, boxplots, world map
- ✅ Key insights mapped to real-world stakeholders

---

## 💡 Key Findings
- Attacks peaked in **2014** driven by ISIS and Taliban
- **Middle East & South Asia** account for 60%+ of incidents
- **Bombing/Explosion** is the most common and deadliest method
- A small number of groups cause the majority of casualties
